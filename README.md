# 🍅 番茄钟 - 商用级专注计时应用

> 基于 Tauri v2 + React 19 + TypeScript 的现代化桌面番茄钟应用

一款精致、高效的番茄钟应用，采用专业的玻璃拟态设计，提供完整的任务管理、数据统计和个性化体验。

## ✨ 核心特性

### 🎯 番茄计时
- **三种模式**: 专注时间 / 短休息 / 长休息，时长可自定义
- **智能切换**: 自动进入下一模式，支持手动切换
- **可视化进度**: 动态进度环，实时显示剩余时间
- **键盘快捷键**: 空格键快速播放/暂停

### 📝 任务管理
- **任务追踪**: 记录每个任务完成的番茄数
- **快速操作**: 添加、完成、删除任务
- **分类显示**: 活跃任务与已完成任务分开展示
- **智能关联**: 自动关联当前活跃任务与番茄钟

### 📊 数据统计
- **每日概览**: 今日番茄数、专注时长、连续天数
- **周度图表**: 可视化展示本周专注趋势
- **累计数据**: 总番茄数、总专注时长统计
- **数据持久化**: 本地存储，永不丢失

### 🎵 环境音效
- **6种白噪音**: 白噪音、粉噪音、棕噪音、雨声、壁炉、森林
- **Web Audio API**: 原生音频合成，无需外部音频文件
- **音量控制**: 独立音量调节
- **一键切换**: 快速开关环境音

### 🎨 个性化
- **8种主题壁纸**: 暖沙、薄雾、花瓣、暮色、石灰、湖蓝、陶土、云白
- **小窗模式**: 160x160 迷你浮动窗口，置顶显示
- **紧凑模式**: 窗口缩小时自动适配
- **透明度调节**: 小窗透明度可调

### 🔧 高级功能
- **数据管理**: 导出/导入 JSON 备份
- **自动控制**: 可设置自动开始下一模式
- **通知提醒**: 完成提示和模式切换通知
- **系统托盘**: 最小化到托盘，快速唤起
- **窗口置顶**: 专注时不被遮挡
- **全局快捷键**: 
  - `Ctrl+1/2/3` - 切换标签页
  - `Ctrl+M` - 小窗模式
  - `Ctrl+T` - 窗口置顶
  - `空格` - 播放/暂停（定时器页面）

## 🎨 设计理念

### 视觉设计
- **玻璃拟态 (Glassmorphism)**: 毛玻璃效果 + 半透明背景
- **柔和配色**: 暖色调为主，护眼舒适
- **流畅动画**: 所有交互都有平滑过渡
- **专业图标**: SVG 矢量图标，清晰锐利

### 交互设计
- **即时反馈**: 按钮 hover/active 状态
- **智能提示**: Toast 通知，关键操作确认
- **键盘优先**: 快捷键覆盖常用操作
- **无损操作**: 数据导出备份，防止误删

### 用户体验
- **零学习成本**: 直观的界面，无需说明
- **高效工作流**: 最少点击完成任务
- **个性化**: 丰富的自定义选项
- **稳定可靠**: 本地存储，离线可用

## 📝 使用技巧

### 番茄工作法最佳实践

1. **设定目标**: 每天开始前添加今日任务
2. **专注工作**: 25 分钟专注，屏蔽干扰
3. **短暂休息**: 5 分钟离开屏幕，放松身心
4. **长休息**: 完成 4 个番茄后，休息 15-30 分钟
5. **复盘总结**: 查看统计数据，改进工作效率

### 推荐设置

- **新手**: 25/5/15 分钟，关闭自动开始
- **资深用户**: 50/10/30 分钟，开启自动开始
- **深度工作**: 90/15/30 分钟，小窗模式

### 环境音推荐

- **专注编码**: 棕噪音、白噪音
- **阅读写作**: 雨声、森林
- **设计创作**: 粉噪音、壁炉

## 🔐 隐私与安全

- **开源透明**: 代码完全开放，可审计
